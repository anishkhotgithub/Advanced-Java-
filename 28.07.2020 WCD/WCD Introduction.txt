Homework:
	
	1) Download all softwares and extract in one folder.
	2) Open Eclipse -> select workspace.
	3) Add tomcat server.
	4) Click on file -> new -> dynamic web project
	5) Give any name to your project -> select tomcat from drop down.
	6) Once project is created -> Right click on webcontent
	7) Create your html files.
